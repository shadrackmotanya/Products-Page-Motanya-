🛍️ Product Listing Website

A simple multi-page product listing website built using HTML, CSS, and JavaScript.

🚀 Features

- Multiple pages (Home, Products, Cart)
- Dynamic product display using JavaScript
- Add to cart functionality
- Cart data stored using localStorage
- Real-time cart count update

🛠️ Technologies

- HTML
- CSS
- JavaScript
